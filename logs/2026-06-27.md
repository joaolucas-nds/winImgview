## Aprendizados / armadilhas
- `<img src=asset://svg>` rasteriza o SVG; é preciso usar `<iframe>` para renderização vetorial — registrado em DEC-005.
- `file:///` é bloqueado pelo CSP do Tauri; protocolo correto é `asset://` via `convertFileSrc()`.
- No Tauri 2, as permissões são declaradas em `capabilities/` (JSON), não no `tauri.conf.json` diretamente.
- argv[1] é como o Explorer passa o arquivo no duplo-clique; o setup() do Tauri é o lugar certo para capturar.
